# typescript vue project

通过koa2引入webpack配置文件进行构建，用babel-loader, ts-loader解析ts文件。前端的server用gulp编译。功能包含了router,vuex,axios。

# 开始项目

### 安装
```shell
npm install
```

### 运行开发环境
```shell
npm run dev
```

### 打开项目地址
```
http://localhost:3030/tsvue/home
```
